What is Hydra?

Hydra is a web-based live-coding environment that allows you to create visuals using JavaScript and WebGL. It is inspired by analog modular synthesizers and early video synthesis. Hydra allows you to create visuals collaboratively in a browser window, and stream them in real time.

There are five types of functions in hydra:
source, geometry, color, blend and modulate

Sources

Hydra uses a few different types of sources to create visuals. These sources can be combined and manipulated in various ways to create complex visuals.

  • solid: solid colors
  • gradient: gradients
  • shape: basic shapes
  • osc: oscillators
  • noise: Perlin noise
  • voronoi: Voronoi diagrams
  • src: output of another layer
  • ...

Colors

Colors in Hydra are represented as RGB values. The values range from 0 to 1, with 0 being the lowest value and 1 being the highest value. You can use the solid source to create solid colors.

Blending

Blending is the process of combining two or more sources together to create a new source. Hydra provides several blending modes that you can use to create different effects.

  • add - adds the values of the sources together
  • diff - subtracts the values of the sources
  • mult - multiplies the values of the sources
  • screen - screens the values of the sources
  • overlay - overlays the values of the sources
  • hardlight - hardlights the values of the sources
  • lighten - lightens the values of the sources
  • darken - darkens the values of the sources
  • difference - calculates the absolute difference between the values of the sources
  • exclusion - similar to difference, but with lower contrast
  • hue - changes the hue of the source
  • saturation - changes the saturation of the source
  • brightness - changes the brightness of the source
  • color - changes the color of the source
  • luminosity - changes the luminosity of the source

Geometry

Hydra provides several functions that you can use to manipulate the geometry of the sources. These functions allow you to scale, rotate, translate, and mask the sources to create different effects.

  • scale - scales the source
  • rotate - rotates the source
  • translate - translates the source
  • mask - masks the source
  • repeat - repeats the source
  • pixelate - pixelates the source
  • modulate - modulates the source
  • modulateHue - modulates the hue of the source
  • modulatePixelate - modulates the pixelation of the source
  • modulateScale - modulates the scale of the source
  • modulateRotate - modulates the rotation of the source
  • modulateKaleid - modulates the kaleidoscope effect of the source
  • modulateRepeat - modulates the repetition of the source
  • modulateScroll - modulates the scrolling of the source
